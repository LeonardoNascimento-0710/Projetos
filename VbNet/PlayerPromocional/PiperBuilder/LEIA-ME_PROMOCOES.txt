ALTERAÇÃO — VÁRIAS PROMOÇÕES POR TEXTO E VOZ POR FRASE
======================================================

ARQUIVOS
--------
SUBSTITUIR:
- Formularios\FrmPrincipal.vb

ADICIONAR:
- PromocaoTexto.vb
- Formularios\FrmPromocoesTexto.vb
- Formularios\FrmPromocoesTexto.Designer.vb
- Formularios\FrmPromocoesTexto.resx

INSTALAÇÃO NO PROJETO
---------------------
1. Faça uma cópia de segurança do projeto.
2. Substitua o FrmPrincipal.vb atual pelo arquivo deste pacote.
3. Copie PromocaoTexto.vb para a pasta raiz do projeto.
4. Copie os três arquivos FrmPromocoesTexto para a pasta Formularios.
5. Abra novamente a solução no Visual Studio.
6. Compile o projeto.

O projeto usa o formato SDK e normalmente reconhece os novos arquivos
automaticamente. Caso não apareçam no Gerenciador de Soluções, clique com o
botão direito no projeto, escolha Adicionar > Item Existente e selecione-os.

COMO FUNCIONA
-------------
- O campo único foi substituído visualmente por uma lista de promoções.
- Clique em "Gerenciar promoções".
- Cada linha possui:
  * texto da promoção;
  * voz Masculina ou Feminina.
- Os botões Subir e Descer determinam a sequência.
- Com a opção aleatória desmarcada, o player segue a ordem das linhas.
- Com a opção aleatória marcada, o player embaralha todas as frases e só
  repete depois que todas tiverem sido reproduzidas.
- "Testar promoção selecionada" reproduz a linha selecionada na lista.
- A configuração antiga é migrada automaticamente para uma promoção masculina.

CACHE DE ÁUDIO
--------------
Cada combinação de texto e voz gera um WAV próprio em:
%LOCALAPPDATA%\PlayerPromocional\Piper\cache_promocoes

Assim o Piper não precisa gerar novamente a mesma frase em toda reprodução.

VOZ FEMININA
------------
O modelo feminino não está incluído neste pacote.

Para instalá-lo:
1. Clique com o botão direito em Instalar_Voz_Feminina.ps1.
2. Escolha "Executar com PowerShell".
3. O script colocará estes arquivos na pasta correta:
   - dii_pt-BR.onnx
   - dii_pt-BR.onnx.json

Pasta esperada:
%LOCALAPPDATA%\PlayerPromocional\Piper\vozes

IMPORTANTE SOBRE LICENÇA
------------------------
A voz feminina indicada é um modelo externo da OpenVoiceOS. Verifique os
termos do modelo antes de incluí-lo no instalador ou redistribuí-lo em uma
versão comercial. O código funciona sem embutir o modelo no executável.
